# Layout

Control display, flexbox, and grid setups easily.

## Flexbox

```html
<div class="sa-flex sa-justify-between sa-items-center sa-bg-gray-100 sa-p-4">
  <div>Left Item</div>
  <div>Right Item</div>
</div>
```

## Grid

```html
<div class="sa-grid sa-grid-cols-3 sa-gap-4">
  <div class="sa-bg-primary-200 sa-p-4">Col 1</div>
  <div class="sa-bg-primary-200 sa-p-4">Col 2</div>
  <div class="sa-bg-primary-200 sa-p-4">Col 3</div>
</div>
```
