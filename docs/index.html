Getting Started

Welcome to the official documentation for the Shoaib Aftab CSS Framework. This framework is designed for enterprise applications demanding 100% security, zero external dependencies, and modern CSS Level 5 features.

<link rel="stylesheet" href="path/to/shoaib-aftab.min.css">

Zero Dependency Philosophy

We strictly forbid the use of external CDNs or Node.js build tools (like PostCSS) for the end-user. The entire framework runs natively in the browser. You don't need Webpack, Vite, or NPM to compile this CSS. It's ready to use out of the box.

GDPR & Native Font Fallbacks

To comply with the strictest European privacy laws (GDPR), we have removed all third-party font calls (e.g., Google Fonts). The framework defaults to ultra-fast native system fonts. If you apply a class like sa-font-inter but the font isn't loaded, it will flawlessly fall back to your OS's native sans-serif font.

Grid & Subgrid

We fully support CSS Grid Level 2, allowing child elements to inherit the grid tracks of their parents using sa-grid-cols-subgrid.

Container Queries

Stop relying entirely on viewport media queries. Use sa-container-type-inline-size on a parent, and then use sa-cq-md:... on children to style them based on the parent's width!

<div class="sa-container-type-inline-size">
  <div class="sa-cq-md:sa-grid-cols-2">...</div>
</div>
